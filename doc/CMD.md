# RESE2NSE Node version 1 Commands

This section lists the supported remote commands for configuration and operation of the node.

- Query commands
    - Power level
    - Aggregator address
    - Sampling period
    - Statistics
    - Control flag
    - Firmware version
- Configure commands
    - Transmit period
    - Power level
    - Radio channel
    - Aggregator address
    - Commit radio settings to NVRAM
    - Node Info: Node ID
    - Node Info: Node Location
    - Control flag
- Operation commands
    - Start Sensing
    - Stop Sensing
    - Timed Sleep
    - Debug Unicast
    - Debug Broadcast
